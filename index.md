# `{abalone}`

# 

## Overview

[abalone](https://jessicabolin.github.io/abalone/) is a package which
documents functions used for defining and mapping climate refugia for
red abalone across California. This project is funded through
California’s 5th Assessment on Climate Change, and supported by the
University of California, Davis.

## Installation

You can install the development version from GitHub via:

    # install.packages("devtools)
    devtools::install_github("JessicaBolin/abalone")

## Usage

Check the [Getting
Started](https://jessicabolin.github.io/abalone/articles/abalone.md)
article to see examples of how to deploy these functions.
